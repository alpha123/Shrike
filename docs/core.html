  • Shrike

    Finds elements matching a CSS selector. Equivilent to the Puma function.

    Arguments

    • A CSS selector String
    • Context to execute the query in DOMElementDOMDocument

    Returns

    All elements matching the CSS selector Array

    Example

              var nodes = Shrike('p.example');

  • Shrike.merge

    Extends an object with properties from other objects.

    Arguments

    • Object to merge the properties of subsequent arguments into Object
    • Objects to merge with the first argument Object
    • Empty function Function to apply to values put into the first object The object whose properties are being copied from The value and key currently being copied Function

    Returns

    The first argument Object

    Example

              var obj1 = {'a': 1}, obj2 = {'b': 2}, obj3 = {'c': 3};
          Shrike.merge(obj1, obj2, obj3, function (value) { return ++value; });
          // obj1: {'a': 2, 'b': 3, 'c': 4}

  • Shrike.extend

    Creates a new object with the properties of all argument objects.

    Arguments

    • Objects to mix the properties of into a new object Object
    • Empty function Function to apply to values put into the new object The object whose properties are being copied from The value and key currently being copied Function

    Returns

    A new object with all the properties of the argument objects

    Example

              var newObj = Shrike.extend({'a': 1}, {'b': 2}, {'c': 3});
          // newObj: {'a': 1, 'b': 2, 'c': 3}

  • Shrike.inherit

    Untangles JavaScript's inheritance mechanism. See http://javascript.crockford.com/prototypal.html. Essentially creates a new object that inherits from the passed object.

    Arguments

    • The object to inherit from Object

    Returns

    An object that inherits from the passed object Object

    Example

              var obj = {hello: function () { return 'Hello'; }},
          newObj = Shrike.inherit(obj);
          newObj.hey = function () { return 'Hey!'; };
          newObj.hello(); // "Hello"

  • Shrike.inspect

    Joins the keys and values of an object into a string.

    Arguments

    • The object to stringify Object
    • ": " The string to separate the idividual keys from values String
    • "\n" The string to separate pairs of keys and values String

    Returns

    A string consisting of the keys and values of the object String

    Example

              var obj = {duck: "quack", cow: "moo", haggis: "haggis sounds"};
          Shrike.inspect(obj); // "duck: quack\ncow: moo\nhaggis: haggis sounds"
